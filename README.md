# Mars Weight Calculator (Python CLI)

**Idea:** A simple Python tool where you enter your Earth weight and get the equivalent weight on Mars, the Moon, and the ISS.

**How it helps NASA/space:** Useful as an educational demo to show gravity differences.

**Technologies:** Python 3

## How to run
1. Clone the repo:  
   `git clone <repo-url>`
2. Navigate to the folder:  
   `cd mars-weight-calculator/cli`
3. Run the script:  
   `python mars_weight.py`
4. Enter your Earth weight when prompted.

## Gravity constants
- Moon: 0.165  
- Mars: 0.38  
- ISS: 0.9
